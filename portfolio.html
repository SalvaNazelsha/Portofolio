Portofolio Saya

Berikut adalah beberapa proyek yang telah saya kerjakan:

Sertifikat Dicoding: Membuat Front-End untuk Pemula

Deskripsi: Proyek ini merupakan kursus yang diambil di Dicoding, di mana saya belajar tentang dasar-dasar pengembangan front-end. Saya membangun sebuah website sederhana menggunakan HTML, CSS, dan JavaScript, yang mencakup navigasi responsif dan elemen interaktif.

Peran: Pengembang Frontend

Hasil: Mampu membuat website yang responsif dan interaktif, serta mendapatkan sertifikat dari Dicoding sebagai bukti penyelesaian kursus.

Sertifikat Dicoding: Pemrograman JavaScript

Deskripsi: Dalam proyek ini, saya mengikuti kursus pemrograman JavaScript di Dicoding. Saya mengembangkan aplikasi web yang memungkinkan pengguna untuk mengelola daftar tugas dengan fitur CRUD (Create, Read, Update, Delete) menggunakan JavaScript dan DOM manipulation.

Peran: Pengembang Full Stack

Hasil: Aplikasi berhasil diimplementasikan dan mendapatkan umpan balik positif dari pengguna, serta meningkatkan pemahaman saya tentang JavaScript dan pengembangan aplikasi web.

Sertifikat Dicoding: Pemrograman Web

Deskripsi: Proyek ini merupakan bagian dari kursus pemrograman web di Dicoding, di mana saya belajar tentang pengembangan aplikasi web menggunakan HTML, CSS, dan JavaScript. Saya mendesain dan mengembangkan sebuah website yang menampilkan portofolio pribadi dengan berbagai fitur interaktif.

Peran : Desainer Grafis dan Pengembang Web

Hasil: Website berhasil diluncurkan dan digunakan sebagai portofolio pribadi, serta mendapatkan sertifikat dari Dicoding sebagai pengakuan atas keterampilan yang diperoleh.
